# Study Time vs Marks Analysis

##  Project Overview
This project analyzes the relationship between study hours and marks using Python.

##  Problem
Students ke study time aur marks ke beech relation samajhna.

##  Solution
Python ka use karke data visualize kiya gaya aur relationship analyze kiya gaya.

##  Tools Used
- Python
- NumPy
- Matplotlib

##  How to Run
1. Python install karo
2. Libraries install karo:
   pip install numpy matplotlib
3. Run command:
   python analysis.py

##  Output
- Scatter plot graph
- Best fit line
- Predicted marks based on input

##  Learnings
- Data visualization
- Linear relationship
- Python programming
